## HW_05. Калькулятор

Создание простого калькулятора с использованием метода __`DOM`__.

> [!IMPORTANT]

## HW_06. Игра "угадайка"

Компьютер определяет число по алгоритму бинарного поиска, при этом сравнение числа с искомым и запуск итераций поиска выполняет пользователь.

> [!IMPORTANT]      
> Обратите внимание! Если в диапазоне будет указано минимальное значение больше/равно максимальному, то выдаст предупреждение.

+ **HTML**
+ **CSS (bootstrap)**
+ **JavaScript**
